# xlaravel

#### 项目介绍
xlaravel是一款基于 laravel6.20.x 和 layui2.5.x 的简单安全的单页面iframe后台管理框架

基于Apache2.0开源协议，可以免费商用二次开发等

其他详细信息请阅读开发文档，文档地址如下
